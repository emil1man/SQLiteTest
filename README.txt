========== ПРЕДИСЛОВИЕ ==========
Задача была реализована на смеси WindowsForm, RDLC, SQLite, ODBC, плюс встроенные в VS17 инструменты для работы с БД. С помощью алгоритмов SQL было создано и связано 3 основные таблицы: "Должности" (хранит в себе название должности и информацию о параметрах зарплаты), "Сотрудники" (хранит имя, должность (ссылкой на первую таблицу) и дату трудоустройства), "Иерархия" (ссылается на Сотрудники по ID и хранит в себе ID начальника для каждого сотрудника. Если у сотрудника нет начальника - значение null), в добавок к ним существует 5 вспомогательных таблиц для вывода информации из запросов.
========== ИНСТРУКЦИЯ ==========
При запуске программа подключаетс к БД "testBD.db" и открывает таблицы "Должности" и "Иерархия". При невозможности подключиться программа закроется с ошибкой. 
Перед пользователем появляется Окно "Form1" с тремя вкладками и рабочей областью.
  Вкладка "Сотрудники":
    Кнопка "Добавить сотрудника": добавляет в таблицу "Сотрудники" запись с заданными именем и должностью, а в таблицу "Иерархия" - ID и указывает, что начальника нет.
    Кнопка "Найти сотрудника по имени": Ищет и выводит записи, где есть совпадения в имени и выводит в текстовую область. 
    
    Записи добавляются в верхнем регистре для облегчения поиска.
    
  Вкладка "Руководители":
    Содержит в себе 2 выпадающих списка, получающих из БД информацию о возможных начальниках (Фильтруя людей с должностью "Employee") и подчиненных (Нету начальника)
    Кнопка "Назначить руководителя": задаст в таблице "Иерархия" выбранного руководителя для выбранного сотрудника.
    Кнопка "Отобразить подчиненных": отобразит для выбранного руководителя всех подчиненных до конца по иерархии в текстовую область.
    
    Обе кнопки срабатывают только при выбранных значениях для выпадающих списков (для "Отобразить подчиненных" хватит только выбранного руководителя).

  Вкладка "Отчёты":
    Содержит в себе 3 кнопки. При нажатии на них открывается диалоговое окно с отчётом, содержащим информацию в соответствии с названием кнопки.
